fb widget - Is a plugin that allows site owners to promote their facebook pages and embed page feed on their websites simply using FB Widget Plugin.


== Description ==
fb widget is a plugin that allows site owners to promote their facebook pages and embed page feed on their websites simply using FB Widget Plugin.


= Features =

    *Easy install by following simple steps
    *Easy to configure.
    *Owner can display his/her facebook page feeds on website
    *Easy display on site using fb widget.
    *Display color schemes.

== Installation ==

Installation consists of following steps:

1. Upload "fb widge plugin" to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the "fb widge" Widget on your sidebar.
4. Your website will rock.

= How to Create a Facebook App =

1. The first thing you need to do is to visit https://developers.facebook.com. In order to create an account.

2. To create a Facebook app, click on Apps on the top of the page and select 'Create a New App'.
	* An display name.This is the app name used when stories are published to Facebook and in a number of other places. 
	* A namespace. Not required for this guide so just leave it blank
	* A category.This will determine the kind of app that you're building.

3. You'll be provided with a dashboard that shows User and API Stats.
